# ⭐ Star Stalker – Catch the Stars, Avoid the Bombs

A fast‑paced arcade game built in pure HTML, CSS, and JavaScript.  
**Single file**, no dependencies, runs instantly in any browser – perfect for hackathons.

![Game Screenshot](https://via.placeholder.com/600x400?text=Star+Stalker+Gameplay)

---

## 🎮 How to Play

- **Goal**: Catch as many ⭐ **stars** as possible while avoiding 💣 **bombs**.
- **Controls**:
  - **Mouse** – Move the basket by moving your cursor over the game area.
  - **Left / Right Arrow Keys** – Move the basket left/right.
- **Scoring**: Each star caught = **+1 point**.
- **Game Over**: Catching a single bomb ends the game.
- **Restart**: Click the **RESTART** button.

> 💡 The game speeds up every 5 points – how high can you score?

---

## ✨ Features

- **Two‑object gameplay** – Stars (good) and bombs (bad).
- **Increasing difficulty** – Speed increases, spawn rate gets faster.
- **Smooth controls** – Mouse tracking + keyboard support.
- **Retro visuals** – Gradient sky, animated basket, hand‑drawn stars and bombs.
- **Responsive canvas** – Works on desktop and laptop.
- **No external libraries** – Pure vanilla JavaScript, runs offline.

---

## 🚀 How to Run

1. **Download** the `star_stalker.html` file.
2. **Double‑click** it – opens in your default web browser.
3. **Start catching stars** immediately – no installation, no server needed.

> 💡 *For best experience, use Chrome, Edge, Firefox, or Safari.*
